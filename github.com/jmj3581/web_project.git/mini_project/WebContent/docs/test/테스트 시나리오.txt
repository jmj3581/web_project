## 테스트를 위한 시나리오 작성
	회원 : 로그인 -> 메인화면의 검색 기능 이용(나라 검색) -> 검색한 나라의 음식 리스트 출력 
		-> 다른 검색도 이용가능 -> 후기작성을 위해 board페이지로 이동 -> 글쓰기를 통해 후기 작성
		(부가적인 기능 : 아이디/비밀번호찾기, 회원가입)

	관리자 : 로그인 -> 회원들을 관리하기 위해 회원리스트 페이지 이동 -> 회원정보 변경/ 조회/ 삭제 가능
		-> 음식점 정보 리스트 페이지 이동 -> 음식점 수정/ 삭제/ 조회 가능
		-> 게시판 리스트 페이지 이동 -> 게시판 글쓰기, 게시글 수정, 게시글 삭제
